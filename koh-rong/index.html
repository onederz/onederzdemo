Finding Your Way to Onederz

From Sihanoukville

By Speedboat: Ferries run daily from Sihanoukville to Long Set Pier (approx. 45 minutes). Onederz is a 5-minute walk from the pier.

From Phnom Penh or Siem Reap

By Bus/Taxi: Travel to Sihanoukville (approx. 4–6 hours), then take a speedboat directly to Long Set Pier on Koh Rong.

Address for Boat Operators

Onederz Koh Rong

Tip: Ask for "Onederz on Long Set Beach" – just a short walk from the pier.
